# itsmehatef/homebrew-tap

Personal Homebrew tap for tools I ship.

## Install

```bash
brew tap itsmehatef/tap
```

## Available

### [stash-mac](https://github.com/itsmehatef/stash-mac)
Tiny macOS menu-bar clipboard history app — Apple Silicon, macOS 14+.

```bash
brew install --cask stash-mac
```

On first launch, grant Stash permission under **System Settings → Privacy & Security → Accessibility** so it can simulate `⌘V` into the previously-active app.
